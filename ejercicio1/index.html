GIT

Git is the open source distributed version control system that facilitates GitHub activities on your laptop or desktop. This cheat sheet summarizes commonly used Git command line instructions for quick reference.

Comando principales de Git

  • cd: Verificamos la carpeta que estamos
  • git INIT: Iniciar un proyecto git.
  • git config --global user.email "Email personal": Identificar y configurar localmente el email en un repositorio local
  • git config --global user.name "name personal": Identificar y configurar localmente el nombre del usuario en un repositorio local
  • git add . : Agrega todos los archivos modificados al repositorio
  • git status: Verificar el estado del repositorio
  • git config user.name Identificar el nombre de usuario del repositorio local
  • git config user.email Verificar el email en el repositorio local
  • pwd Muestra la ruta al directorio actual
  • ls Muestra la lista de contenidos de un directorio
  • clear Limpiar pantalla del terminal
  • git config -- global --unset-all user.name Elimina las configuraciones del nombre de usuario en el repositorio local
  • cd nombre-directorio Ingresa a un directorio(carpeta) en la terminal
  • cd .. Salir del directorio en la terminal
  • git log Mostrar el historial de cambios en el repositorio
  • cd ..
